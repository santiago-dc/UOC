PREGUNTA 3: Aplica de nuevo el modelo anterior, pero usando una mtrica distinta y compara los resultados.

DBSCAN: es un mtodo de clustering no supervisado muy conocido. La mtrica en la que se basa es la densidad de datos, es decir, calcula los grupos basndose en la densidad de datos que hay en las secciones del hiperplano. Si hay 2 grupos, buscar dos puntos con alta densidad y los considerar los centroides.

In [186]:
from sklearn.cluster import DBSCAN
model = DBSCAN(eps=0.05, min_samples=10)
yhat = model.fit_predict(X)
fun=lambda x: 0 if x==0 else 1
vfun = np.vectorize(fun)
accuracy_score(y, vfun(yhat))
Out[186]:
0.9633967789165446

Con los parmetros adecuados (eps para distancia mnima entre mxima entre miembros del mismo grupo y Min_samples para el mnimo por grupo). Observamos una precisin muy buena, a la altura del kmeans y sin tener que indicar el numero de clusters. Esto es porque, para este caso los puntos estn bastante bien distribuidos, dejando pocos datos aislados lejos de los centroides, que es dosnde suele haber fallos en este algoritmo.

In [187]:
pca_data["target"] = vfun(yhat)
sns.scatterplot(x = "p1", y = "p2", hue = "target", data = pca_data)
plt.title("DBSACN clusters")

Quizs este plot pueda contrastar con lo dicho anteriormente, esto se debe a que los puntos entre los dos centroides son los que dan problemas a ambis mtodos, mientras kmeans los clasifica todos como 0, DBSCAN lo hace como 1.

PREGUNTA 6: Ha habido mejora en capacidad predictiva, tras aplicar PCA/SVD? A qu crees que es debido?.

El resultado ha mejorado, esto puede ser, debido a que los dos elementos extraidos usando PCA, aunque no recogen toda la informacin, al 100%, extraen caractersticas de los datos que son ms determinantes que las que no recogen.

El PCA es una tcnica muy prctica para reducir la dimensionalidad de nuestro set de datos sin perder mucha informacin. En nuestro caso el entrenamiento se muestra igual, pero el test ha mejorado un 5% poco.

Considero que el PCA es una tcnica esencial a tener en cuenta como Data Scientist sobretodo en problemas de clasificacin. Es sencilla de implementar, si da mejores resultados o parecidos puedes usarla, pues mejora el rendimiento del modelo. Si empeora, puedes utilizarla para la visualizacin de los resultados, como en apartados anteriores.
